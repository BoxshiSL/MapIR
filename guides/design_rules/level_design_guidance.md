# Level design + player guidance

Curated from "Thesis · Level Design — Resources and techniques to guide the
player" and "Основы Дизайна Уровней Руководство для Начинающих." in
`guides/`.

## Player guidance rules

* **guidance_leading_lines** — roads, shadows, silhouettes, light contrast
  all point toward the next focal point.
* **guidance_breadcrumbs** — small repeated cues imply a path without HUD
  markers.
* **landmarks_one_per_district** — landmarks reduce confusion and create
  scale (also referenced from urban-playgrounds).

## Iteration

* **iteration_throwaway_blockouts** — first passes are wrong on purpose.
  Playtest and iterate before detailing.
* **planning_blockout_first** — see urban-playgrounds note.

## Player agency / readability

* Diegetic guidance (light, colour, sound) is usually more immersive than
  HUD markers.
* Negative space isolates the subject — useful for landmark composition.
* Affordances + signifiers clarify which objects/paths can be used.
